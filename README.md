# Doom Loop Simulator

This is a simple web app simulating the "Doom loop", i.e. a negative feedback spiral between a country's government (sovereign) and its domestic banking system. Developed for my students for educational purposes. 

## How to use

1. Clone the repository (or download the html file)

2. Open the index.html file in your browser

3. Adjust the parameters in the left panel

4. Click the "Run Simulation" button

5. Observe the results in the right panel

6. Click the "Reset" button to start over

## Prameters
- **Resolution** (toggle on or off) Represents the effect of a Single Resolution Mechanism (SRM)/Bail-in 
- **Risk Sharing** (toggle on or off) Represents the effect of risk sharing mechanisms such as a European Deposit Insurance Scheme (EDIS)

## Results

- **Sovereign Creditworthiness**: Represents the sovereign's creditworthiness over time
- **Banking Sector Capital**: Represents the bank's capital over time
- **Real Economy Impact**: Represents the impact of the doom loop on the real economy over time
